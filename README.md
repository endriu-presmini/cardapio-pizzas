# 🍕 Cardápio de Pizzas

Cardápio digital responsivo desenvolvido com HTML e CSS puro, exibindo pizzas em cards com imagem, descrição e preço.

## 🚀 Tecnologias

- HTML5
- CSS3 (Flexbox, media queries, object-fit)

## ✨ Destaques

- Layout responsivo (desktop e celular)
- Cards com imagem, nome, descrição e preço
- Botão de adicionar ao carrinho estilizado
- Adaptação automática para telas pequenas com media query
